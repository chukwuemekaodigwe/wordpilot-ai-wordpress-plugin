WordPilot helps users post AI-generated, SEO-rich content directly to their WordPress sites.

== Description ==

WordPilot is a powerful WordPress plugin that seamlessly integrates with the WordPilot AI platform, allowing users to effortlessly publish the AI-generated, SEO-optimized content from WordPilot directly to their WordPress sites.

Key Features:
* Direct posting of AI-generated content to WordPress
* SEO-rich content optimization
* Page view and visit counting
* Custom dashboard for content management
* Secure API integration with WordPilot AI platform

= How It Works =

1. Generate content using the WordPilot AI platform
2. Connect your WordPress site to WordPilot
3. Publish AI-generated content directly to your site
4. Monitor performance with built-in analytics

Visit WordPilot dashboard: https://app.wordpilot.ai/

== Installation ==

1. Upload the `wordpilot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the WordPilot settings page to connect your account

== Frequently Asked Questions ==

= Do I need a WordPilot AI account to use this plugin? =

Yes, you need an active WordPilot AI account to generate and publish content using this plugin.

= Is the plugin compatible with my theme? =

WordPilot is designed to be compatible with most WordPress themes. If you encounter any issues, please contact our support team.

= How secure is the connection between my site and WordPilot AI? =

We use industry-standard security practices, including API authentication and secure data transfer protocols, to ensure the safety of your data.

= Can I reconnect my Wordpilot dashboard to my wordpress if I make an update ? =

Yes you could reconnect your wordpress site to your WordPilot dashboard by going to the Wordpress plugin page and clicking on the "Reconnect" button. and proceed to WOrdPilot to edit the site link and copy verification back to your wordpress site to complete the reconnection.
